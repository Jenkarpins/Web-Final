About Achievement Wiki

A web app for tracking video game achievements with a sleek, Steam-inspired interface.

Key Features

  • Steam-style dark theme with dynamic progress bars
  • Achievement tracking with checkboxes
  • Game-specific pages with JSON-loaded data
  • Clean sidebar navigation and game previews

Technologies Used

  • HTML / CSS
  • JavaScript
  • JSON for game data
  • GitHub Pages for hosting

Project Purpose

This website was made for my ICS 200 class. It was inspired by Steam's design while trying to maintain a Wiki feel.
